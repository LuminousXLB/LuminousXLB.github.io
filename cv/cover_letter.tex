    %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
    \lettersection{Research Interest}
    %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

    My research interest is in Computer Security, especially in Confidential Computing currently.

    In the context of outsourced computing, confidential computing focuses on isolating data in hardware-enforced secure enclaves.
    It is based on Trusted Execution Environment, such as Intel Software Guard Extensions (Intel SGX).
    By leveraging dedicated hardware or hardware extension, confidential computing can provide security guarantees regarding the confidentiality of sensitive data and the integrity of computing results.
    Besides the basic security principles, the emerging technology can also improve current systems in terms of security or performance.

    Some existing work has concentrated on the security problems of TEE itself.
    There are classical vulnerabilities transferred to the TEE environment, such as memory error and corresponding defences.
    There are side-channel problems, such as privacy leakage due to access pattern, that can be analogue to similar problems in cloud storage.
    There are also new challenges against TEE itself, such as the security problem about the attestation mechanism.

    Others have built some conceptual or practical systems based on the security guarantee provided by TEE.
    For instance, researchers have designed a bitcoin wallet that preserves better privacy, secure resource accounting service on computation platforms, and trusted network middleboxes.
    TEE has been included in the building of blockchain in the Microsoft Confidential Consortium Framework as well.
